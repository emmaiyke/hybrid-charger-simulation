# Hybrid-Charging System
This project simulates a power system that maintains a stable 5V output (±0.5V) with up to 95% conversion efficiency using 12V battery, 14.4V solar, and 15V AC inputs. It features a smart voltage regulator that dynamically corrects fluctuations, tracks voltage deviations, correction factors, and error history with graph visualization. An efficiency optimizer evaluates voltage matching, load conditions, and thermal effects in real time. A thermal analysis module monitors temperature from a 25°C baseline, estimates heat losses, prevents overheating and thermal stress, and ensures long-term reliability and optimized performance.
